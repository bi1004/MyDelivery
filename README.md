## 💭 URL 만들기

🍕 음식 등록: POST /foods
localhost:8080/foods

🍤 음식 전체 조회: GET /foods
localhost:8080/foods

🍪 음식 단일 조회: GET /foods/{food_id}
localhost:8080/foods/1

🌭 음식 수정: PUT /foods/{food_id}
localhost:8080/foods/1

🥩 음식 삭제: DELETE /foods/{food_id}
localhost:8080/foods/1
